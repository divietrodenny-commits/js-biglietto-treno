-Richiesta dati all’utente
  -Si chiede all’utente quanti chilometri vuole percorrere.
  -Si chiede all’utente quanti anni ha il passeggero.

-Calcolo del prezzo base
  -Si calcola il prezzo base moltiplicando i chilometri per 0.21 euro al km.

-Applicazione degli sconti in base all’età
    SE l’età è minore di 18 anni
       -si applica lo sconto del 20% sul prezzo base

ALTRIMENTI 
    SE l’età è maggiore o uguale a 65 anni
       -si applica lo sconto del 40% sul prezzo base

ALTRIMENTI
       -non si applica nessuno sconto

-Far vedere il risultato final all'utente.
